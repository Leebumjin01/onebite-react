- useMemo
  "메모이제이션" 기법을 기반으로 불 필요한 연산을 최적화 하는 리액트 훅

- React.memo
  컴포넌트를 인수로 받아, 최적화된 컴포넌트로 만들어 반환
  부모 컴포넌트가 리렌더링 되더라도 자신이 받는 props가 바뀌지 않으면 리렌더링이 발생되지 않음(불 필요한 리렌더링 방지)

-- 최적화는 기능을 먼저 구현 후 진행
-- 함수를 많이 갖고 있어 무거운 컴포넌트를 최적화
